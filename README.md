# Elavon

A payment processing library specifically for Elavon in Elixir

## Status

This is a work in progress. It has plenty of patches of code here and there.

## Installation

```elixir
def deps do
  [
    {:elavon, "https://github.com/auroche/elavon.git"}
  ]
end
```
